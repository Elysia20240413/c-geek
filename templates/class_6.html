break 语句

break 语句简介

break 语句用于立即终止循环，并继续执行紧接着循环的下一条语句。

示例：break 语句


#include <stdio.h>

int main ()
{
    for( int a = 10; a < 20; a++ )
    {
        if( a == 15 )
        {
            break;
        }
        printf("a 的值： %d\n", a);
    }

    return 0;
}

上面的代码在 a 等于 15 时终止循环，只打印 10 到 14 的 a 的值。
